# WS01_HTML – Websivuston kehittäminen

Tämä repositorio on Laurea-ammattikorkeakoulun **Websivuston kehittäminen** -opintojaksoa varten. Repositorioon kootaan harjoitukset ja mahdolliset projektityöt opintojakson aikana.

## Tavoitteet

Opintojakson suoritettuaan opiskelija osaa:

- suunnitella, toteuttaa ja julkaista verkkosivustoja ja web-pohjaisia käyttöliittymiä alan keskeisiä työvälineitä ja standardeja käyttäen
- arvioida websivustojen toiminnallisuutta, saavutettavuutta ja kehittämistarpeita

## Repositorion käyttö

- Jokainen tehtävä tehdään omaan tiedostoonsa tai kansioonsa.
- Muutokset tallennetaan selkeillä commit-viesteillä.
- README:tä päivitetään tarvittaessa, kun tehtävärakenne laajenee.

## Opintojakson moduulit

Opintojakson sisältö koostuu seuraavista moduuleista:

- WS01_HTML
- WS02_CSS
- WS03_CSS_Advanced
- WS04_Page_Layout
- WS05_Responsive_Design
- WS06_Bootstrap
