Page Layout

CSS provides two modern layout systems for building page structures.

Semantic elements

  • <header> – site or section header
  • <nav> – navigation links
  • <main> – primary page content
  • <aside> – sidebar / supplementary content
  • <footer> – site or section footer

CSS Flexbox

One-dimensional layout — rows or columns. Used here for the navbar.
