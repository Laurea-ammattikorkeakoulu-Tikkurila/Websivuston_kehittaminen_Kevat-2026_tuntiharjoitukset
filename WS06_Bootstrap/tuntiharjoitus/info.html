What is Bootstrap?

Open source v5.3 Mobile-first

Bootstrap is the world's most popular front-end framework. It gives developers a ready-made toolkit of HTML, CSS and JavaScript components so they can build responsive, accessible websites without writing everything from scratch.
Without Bootstrap
  • Write all CSS manually (WS02–WS05)
  • Build your own responsive grid with CSS Grid / Flexbox
  • Create hamburger menus with CSS checkbox hacks
  • Style every form input individually
  • Test across browsers for inconsistencies
With Bootstrap
  • 12-column responsive grid out of the box
  • Pre-built navbar with working hamburger (JS)
  • Consistent form styling with one class
  • Cross-browser tested by the Bootstrap team
  • Hundreds of utility classes replace custom CSS

Bootstrap Key Concepts

Click each heading to expand it.

Bootstrap was created by Mark Otto and Jacob Thornton at Twitter in 2011, originally to encourage consistency across internal tools. It was open-sourced in August 2011 and quickly became the most popular front-end framework in the world.

  • Bootstrap 2 (2012) — added responsive features
  • Bootstrap 3 (2013) — mobile-first redesign
  • Bootstrap 4 (2018) — switched from Less to Sass, added Flexbox grid
  • Bootstrap 5 (2021) — dropped jQuery, added utility API, updated components

Bootstrap's grid divides the page into 12 equal columns. You choose how many columns each element uses at each breakpoint:

Class prefix Breakpoint Screen width
col- Extra small < 576 px
col-sm- Small ≥ 576 px
col-md- Medium ≥ 768 px
col-lg- Large ≥ 992 px
col-xl- Extra large ≥ 1200 px
