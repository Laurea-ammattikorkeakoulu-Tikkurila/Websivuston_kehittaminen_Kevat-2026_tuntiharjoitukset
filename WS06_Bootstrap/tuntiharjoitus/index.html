Building responsive websites fast with the Bootstrap 5 framework

Learn about Bootstrap

What we cover in WS06

1Bootstrap Grid

Replace hand-written CSS Grid with Bootstrap's 12-column container / row / col-* system. Columns respond automatically to the screen size using breakpoint prefixes.

Classes: container, row, col-md-4
2Navbar & Components

Bootstrap's navbar component replaces the WS05 CSS-only checkbox hamburger. The JS bundle handles toggling the mobile menu with full accessibility support.

Classes: navbar, navbar-toggler, collapse
3Utility Classes

Bootstrap ships hundreds of utility classes for spacing, colour, typography and layout. Most styling in this exercise is done with these classes — not custom CSS.

Classes: mt-*, py-*, text-center, d-flex
