# WS03_CSS_advanced - CSS Edistyneet Ominaisuudet

Tämä on työpaja WS03_CSS_advanced kansio, joka keskittyy CSS:n edistyneisiin ominaisuuksiin.

## Sisältö

Tässä työpajassa käsitellään seuraavia CSS:n edistyneitä ominaisuuksia:

### 1. CSS Transitions (Siirtymät)
CSS transition-ominaisuuden avulla voidaan luoda tasaisia animaatioita elementin ominaisuuksien muutoksista. Käsitellään transition-duration, transition-timing-function ja muut transition-liittyviä ominaisuuksia.

### 2. CSS Animation (Animaatiot)
Monimutkaisempien animaatioiden luominen keyframe-sääntöjen avulla. Käsitellään animation-duration, animation-iteration-count, animation-direction ja muut animation-liittyviä ominaisuuksia.

### 3. Pseudo-classes (Pseudo-luokat)
Pseudo-luokkien käyttö elementtien eri tilojen tyylittelyyn, kuten :hover, :focus, :active, :nth-child(), :first-child, :last-child ja muut pseudo-luokat.

### 4. CSS Filters (Suodattimet)
CSS filter-ominaisuuden avulla voidaan soveltaa visuaalisia efektejä, kuten blur, brightness, contrast, grayscale, hue-rotate ja muut suodattimetyypit.

### 5. Advanced Selectors (Edistyneet Selektorit)
Monimutkaisten selektoreiden käyttö, kuten kombinaator-selektorit (>, +, ~), attribuuttiselektorit, pseudo-elementit ja niiden yhdistelmät.
